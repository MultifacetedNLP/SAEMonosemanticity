# Evaluating Sparse Autoencoders for Monosemantic Representation

Welcome to the repository for **"Evaluating Sparse Autoencoders for Monosemantic Representation"**.  

## Requirements

### Prerequisites
To run this code, ensure that you have:
- Python 3.10.13 installed on your system.

### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/MultifacetedNLP/SAEMonosemanticity.git
   cd SAEMonosemanticity
   ```
2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```
